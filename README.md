# FSD_Assignment1 : Music App

A modern, responsive music streaming application built with React and Vite. Stream your favorite music, create playlists, and enjoy a seamless listening experience.

## Features

- Music player with play, pause, skip, and volume controls
- Browse and search through music library
- Create and manage favorite playlists
- Search functionality to find your favorite tracks
- Modern and responsive UI with smooth animations
- Fast and optimized performance with Vite

## Technologies Used

- React 19
- React Router v7
- Vite
- CSS3
- React Icons
- ESLint for code quality

## Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher) or yarn

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/music-app.git
   cd music-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   The app will be running at [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
# or
yarn build
```

## Running Tests

```bash
npm test
# or
yarn test
```
